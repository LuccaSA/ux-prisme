Icônes

Vous pouvez générer une icône grâce à la mixin makeIcon.

@import "~@lucca-front/icons/src/mixins";
@include makeIcon("heart");
info
Il est très souvent utile d'utiliser la mixin makeIcon dans un ::before

Ombrages

Vous pouvez ajouter de l'ombre à un composant grâce à la mixin elevated(1~5).

@import "~@lucca-front/scss/src/mixins";
@include elevate(1);

Loader

Vous pouvez ajouter un loader via la mixing loading.

@import "~@lucca-front/scss/src/mixins";
@include loading($size);

Media queries

Vous pouvez utiliser les breakpoints de votre thème dans vos media queries :

@import "~@lucca-front/scss/src/mixins";
@include media_smaller_than("nom_breakpoint") {}
@include media_larger_than("nom_breakpoint") {}
@include media_between("nom_breakpoint1", "nom_breakpoint2") {}

Masquage

Vous pouvez utiliser la mixin mask pour masquer visuellement un élément sans pour autant le retirer de la couche sémantique du document.

@import "~@lucca-front/scss/src/mixins";
@include mask;
info
La classe .u-mask fait directement appel à cette mixin.

IE 11

La mixin ie11 permet de filtrer ce navigateur tout en étant explicite dans votre code malgré le hack CSS.

@import "~@lucca-front/scss/src/mixins";
@include ie11 { … }
