Les fonctions sont accessibles en ajoutant @import "~@lucca-front/scss/src/theming.overridable" à votre composant.
Elles retournent des objets dont la notation est parent.child.[...].

Couleurs & Palette

Pour accéder à une couleur de votre palette :

_color("nom_couleur") ex. _color("primary")

Pour accéder à une valeur secondaire de cette couleur (texte, déclinaison...) :

_color("nom_couleur", "clé") ex. _color("primary", "text")

Composants

Pour accéder au thème des composants :

_component("nom_composant.clé.[...]") ex. _component("button.padding")

Theme

Pour accéder de manière générique aux maps du thème :

_theme("nom_map.clé.[...]") ex. _theme("commons.background.base");

Get

Pour accéder aux valeurs de votre propre map :

_get($nom_map, "clé.[...]")
